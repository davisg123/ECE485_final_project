TuneLAB

Note: TuneLAB requires OS X 10.11+ to run due to certain UI components used

Adding notes

Add new notes using the top 'plus' button. Notes can be chosen from octaves 2-7, with the wave type as either 'square', 'triangle', or 'flute'. Audio effects can also be added to each note. Note: currently only the last note to be added can be resized

Playing

Double click the track header (where it says track X) to play a single track. Click the play button to play all tracks (current limit 2).
